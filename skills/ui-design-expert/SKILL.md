# 秒悟美术UI UE设计专家

## 任务目标
- 本 Skill 用于：根据用户描述的 Web 应用需求，智能推荐最合适的设计风格和UI方案
- 能力：
  - 分析用户需求并匹配最佳设计风格
  - 提供完整的设计系统规范（色彩、字体、间距、组件）
  - 生成 shadcn/ui 组件使用指南
  - 输出可直接落地的设计代码片段
- 触发：当用户需要UI设计建议、设计系统、界面风格推荐、shadcn/ui实现时

## 前置准备
- 无需额外依赖，纯智能体能力实现

## 操作步骤

### 1. 需求分析
- 仔细阅读用户描述的 Web 应用需求
- 提取关键信息：
  - 应用类型（SaaS、电商、博客、仪表盘、社交等）
  - 目标用户群体（B端/C端、年龄层、专业度）
  - 核心功能模块
  - 品牌调性偏好（现代/传统、活泼/沉稳、简约/丰富）

### 2. 设计风格匹配
- 从设计库中选择最匹配的 3-5 种风格
- 每种风格提供：
  - 风格名称和特点描述
  - 适用场景说明
  - 视觉关键词（如：极简、玻璃态、新拟态等）
  - 推荐配色方案

### 3. 设计系统生成
- 为主推风格生成完整设计系统：
  - **色彩系统**：主色、辅助色、中性色、语义色（成功/警告/错误/信息）
  - **字体系统**：字体家族、字号层级、行高、字重
  - **间距系统**：基础单位、组件间距、布局间距
  - **圆角系统**：组件圆角规范
  - **阴影系统**：层级阴影定义

### 4. shadcn/ui 实现指南
- 提供 shadcn/ui 组件使用建议：
  - 推荐安装的基础组件列表
  - 主题配置代码（tailwind.config.ts）
  - CSS 变量定义（globals.css）
  - 常用组件代码示例

### 5. 页面结构建议
- 根据应用类型提供页面布局建议
- 关键组件组合推荐
- 响应式设计要点

## 设计风格库

### 现代极简风格
- **特点**：大量留白、简洁线条、单色或双色配色
- **适用**：SaaS产品、企业官网、工具类应用
- **关键词**：Minimalist、Clean、Whitespace
- **推荐配色**：白/灰/黑 + 单一品牌色

### 玻璃态风格
- **特点**：半透明背景、模糊效果、层次感强
- **适用**：创意展示、个人作品集、现代仪表盘
- **关键词**：Glassmorphism、Blur、Translucent
- **推荐配色**：渐变背景 + 半透明白色卡片

### 新拟态风格
- **特点**：柔和阴影、凸起/凹陷效果、柔和配色
- **适用**：音乐播放器、智能家居、创意工具
- **关键词**：Neumorphism、Soft UI、Skeuomorphism
- **推荐配色**：浅灰背景 + 同色系阴影

### 暗黑科技风
- **特点**：深色背景、霓虹点缀、高对比度
- **适用**：开发者工具、游戏、科技产品
- **关键词**：Dark Mode、Cyberpunk、Neon
- **推荐配色**：深灰/黑 + 荧光绿/蓝/紫

### 温暖有机风
- **特点**：圆角元素、大地色系、自然质感
- **适用**：生活方式、健康、教育类应用
- **关键词**：Organic、Warm、Earthy
- **推荐配色**：米色、棕色、柔和绿色

### 粗野主义风格
- **特点**：大胆排版、高对比色、打破常规
- **适用**：创意机构、艺术展示、品牌官网
- **关键词**：Brutalism、Bold、Raw
- **推荐配色**：黑/白 + 高饱和强调色

### 企业专业风
- **特点**：稳重配色、清晰层级、数据可视化
- **适用**：B端产品、金融、数据分析
- **关键词**：Corporate、Professional、Data-driven
- **推荐配色**：深蓝、灰色、白色
