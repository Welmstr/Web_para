## 使用方式
当用户提到某个品牌或风格时，读取对应的 resources/<brand>.md，
理解其设计语言后应用到 UI 生成、设计建议或 DESIGN.md 创作中。
支持组合风格，例如"Stripe 的排版 + Notion 的色调"。
